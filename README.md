# ModerHLPR-v2

Приветствую, уважаемые модераторы!
Сделал небольшое дополнение для браузера, чтобы упростить и ускорить процесс модерации. Вот краткий список функций, которые оно добавляет:

Динамический перевод чата: Автоматически переводит сообщения в блоке «История Чата» (кроме русского и украинского) прямо на странице, но только тогда, когда они появляются на экране. Благодаря этому сайт не лагает и не зависает на больших тикетах. Скрипт работает только в окне тикета \ репорта.

Панели быстрых ответов: Над текстовыми полями появляются удобные кнопки для моментальной вставки готовых причин наказаний в тикетах и предупреждений в окнах уведомлений игрока.

Умные рекомендации: Скрипт сам сканирует чат за последние 24 часа на наличие жестких нарушений/банвордов и сразу выводит подсказку над полем ввода: какую причину и срок мута рекомендуется выдать. Также есть встроенная проверка на дубликаты, чтобы случайно не выдать мут за одно и то же сообщение дважды.

Авто-копирование никнейма: При нажатии на кнопку «Принять тикет» ник нарушителя сразу копируется в буфер обмена.

Кликер для IP: Все IP-адреса на странице становятся кликабельными — можно помечать их как рабочие (зеленый), статус сохраняется.

Автообновление: Скрипт сам каждые 30 секунд обновляет блоки текущего сервера, онлайна и чата, чтобы данные всегда были свежими без ручной перезагрузки страницы.
